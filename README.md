# Tung Shing MCP Server

黄历 MCP

## Usage

### npx

```json
{
  "mcpServers": {
    "tung-shing": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-tung-shing"
      ]
    }
  }
}
```

### cli

```bash
npm install -g mcp-tung-shing
```
